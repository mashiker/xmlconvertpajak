Setup OpenClaw: Langkah 1-3

Persiapan sampai OpenClaw pertama kali jalan

1

Install Node.js

Download Node.js versi LTS dari nodejs.org. Pilih versi 20.x atau lebih baru.

Pastikan npm sudah terpasang:

node --version

npm --version

2

Install OpenClaw

Buka terminal, lalu jalankan perintah instalasi resmi:

curl -fsSL https://get.openclaw.ai | bash

Atau via npm:

npm install -g openclaw

3

Onboarding

Jalankan perintah pertama kali:

openclaw

OpenClaw akan otomatis guide kamu setup API key (Claude/GPT) dan konfigurasi awal.
