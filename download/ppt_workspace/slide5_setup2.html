Setup OpenClaw: Langkah 4-5

Sambungkan ke platform dan mulai pasang skill

4

Hubungkan Platform

Sambungkan ke WhatsApp, Telegram, atau Discord:

openclaw connect whatsapp

openclaw connect discord

Ikuti QR code atau token auth yang muncul di layar.

5

Pasang Skill Pertama

Cari dan install skill dari ClawHub:

openclaw skill install web-search

openclaw skill install pdf

openclaw skill install gog

Tip: Pasang skill GOG (Google Workspace) dulu, ini paling berguna buat anak pajak!
